# Linked list

- Linked list are physically represented in memory
- Each node is made of two parts: The data, and a pointer to the next node.
- The first node is the **head** and the last node is the **tail**.

## Time complexity 
<br>

| Action | Average | Worst |
| ------ | ------  | ----- |
| Search | Θ(n) | O(n)|
| Insert | Θ(1) | O(1)|
| Delete | Θ(1) | O(1)|
